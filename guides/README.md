# PCL-Lite — Usage Guides

This folder contains **short guides** to help you use PCL-Lite effectively: standard (general), image, and video bootstraps.

| Guide | Bootstrap | Use case |
|-------|-----------|----------|
| [**Standard**](./guide-standard.md) | [bootstrap_lite_en.md](../bootstrap_lite_en.md) / [bootstrap_lite_fr.md](../bootstrap_lite_fr.md) | Architecture, development, DevOps, teams, general personas |
| [**Image**](./guide-image.md) | [bootstrap-image/](../bootstrap-image/) | Visual design, illustrations, brand assets, accessibility, print, social visuals |
| [**Video**](./guide-video.md) | [bootstrap-video/](../bootstrap-video/) | Scripts, storyboards, editing, captions, motion, delivery specs |

---

## How to use a guide

1. Choose the bootstrap that matches your task (standard, image, or video).
2. Open the corresponding guide and the bootstrap file (same language: EN or FR).
3. Copy the **entire bootstrap** into your AI chat (first message).
4. Wait for confirmation (e.g. *"PCL-Lite … Ready"*).
5. Follow the guide: use `/persona` commands and examples.

---

## Where to run PCL-Lite

- **Chats**: ChatGPT, Claude, Gemini, DeepSeek, Cursor — paste the bootstrap, then use `/persona` commands.
- **Image/Video**: The bootstrap outputs **text** (prompts, specs, scripts). For actual image or video generation, use your usual tools after getting the specs from the chat.

See each guide for platform tips and workflow.
